		
		[README]

Program ini memiliki dua fungsi utama yaitu mencari akar kuadrat (sqr) dan aka persamaan kuadrat (apk)
Untuk menjalankan fungsinya, program ini menggunakan command line
1. (sqr) Untuk mencari akar gunakan argumen sqr "./main sqr" kemuadian masukkan angka yang akan dicari nilai akarnya.
2. (apk) Untuk mencari akar persamaan kuadrat gunakan argumen apk "./main apk" 
kemudian masukkan koefisien dari AX^2, BX dan C yang dipisahkan dengan spasi secara berurutan.
